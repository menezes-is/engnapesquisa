WoodpeckeR



Programa de Dimensionamento de Vigas de Madeira




WoodpeckeR foi projetado de modo a ensinar e auxiliar pessoas com interesse em dimensionar vigas de madeira sujeitas a flexão simples, ou seja, determinar as dimensões necessárias para que uma viga de madeira serrada possa ser utilizada com segurança.

Por meio dele, é possível realizar diversas verificações, sendo elas:

Resistência de Cálculo
Flexão Simples
Cisalhamento
Compressão Normal às Fibras
Estabilidade Lateral de Viga
Estados Limites de Serviço

Todos os cálculos do programa são realizados de acordo com as normas técnicas brasileiras, especificamente a ABNT NBR 7190-1:2022.


WoodpeckeR foi projetado de modo a funcionar de maneira intuitiva, no início, cada página apresenta uma breve introdução do assunto sendo tratado. Se o usuário entendar, poderá seguir em frente para a seção na qual informará os parâmetros solicitados. Do contrário, será direcionado a uma explicação mais aprofundada através de botões.

Após o preenchimento dos parâmetros, o usuário será alertado caso algo não seja preenchido, deve-se selecionar o botão Salvar Valores, para assim liberar a seção de respostas, que apresentará diversos botões, cada um com uma verificação específica.

se desejar realizar uma nova verificação basta repetir o processo, informar os novos valores e selecionar o botão Salvar Valores.

Também nessa seção, é possível encontrar o botão Emitir PDF. Através dele será possível obter um PDF que mostrará, em detalhes, como os resultados disponibilizados ao usuário foram obtidos.

Por fim, após as verificações, será possível encontrar um link que o levará até a próxima etapa do programa.
