O Estado Limite de Serviço é o parâmentro que define se a estrutura atende os critérios de durabilidade, aparência, conforto e utilização funcional de maneira adequada.

As informações necessárias nesta página são:
No caso de flexão reta é necessário apenas o valor das flechas instantânea e final.
Na flexão olbiqua é necessario uma flecha de cada para o eixo x e para o eixo y.

As contas realizadas nessa página são baseadas nas informações dadas anteriormente nas outras páginas.
